# Linux Network Skills Lab

Practical Linux networking notes, commands, labs, and troubleshooting playbooks for DevOps, Cloud Engineering, SRE, and Security Operations.

## Purpose

This repository is used to document and practice Linux networking fundamentals with sanitized examples.

The focus is on practical troubleshooting skills used in real server and cloud environments.

## Skills Covered

- Basic networking fundamentals
- OSI model simplified view
- IP address and private IP ranges
- Subnets and CIDR basics
- Gateway and routing basics
- DNS basics
- TCP and UDP basics
- Network ports
- Firewall basics
- VPC, public subnet, private subnet
- Load balancer basics
- Network interface inspection
- Routing table inspection
- Listening port inspection
- Connectivity troubleshooting

## Security Note

All command outputs in this repository are sanitized examples.

Do not publish real:

- Public IP addresses
- Private IP addresses from company networks
- MAC addresses
- Hostnames
- Usernames
- Internal domains
- Server names
- Company network details
